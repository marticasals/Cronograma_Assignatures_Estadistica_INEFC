# Cronograma de les assignatures d'Estadística i Anàlisi d'Esports a l'INEFC Barcelona

En aquest repositori l'alumnat pot consultar, des del primer dia de classe, el cronograma actualitzat de les assignatures.

Aquest espai permet una planificació transparent del curs i facilita el seguiment dels continguts, activitats i avaluacions.
